# HydroLab Alpha

Экспериментальная платформа для исследования, мониторинга и управления гидропонными системами.

## Обзор

**HydroLab Alpha** — дальнейшее развитие ESPHydro. Теперь это не просто гидропонная установка на одну систему - это блок управления - гидронная лаборатория на 255 одновременно подключенных установок.
Из ESPHydro перешли все продвинутые режимы EBB, установлены новые industry-grade датчики для высокоточного сбора информации, развернуто клиент-серверное приложение для удобного доступа к данным и управлением системами.

Система объединяет:

* датчики
* исполнительные механизмы
* сбор данных
* аналитику
* веб-интерфейс

в единый замкнутый контур.

## Идея проекта

* Блок гидропонной установки с продвинутыми датчиками. Не только выращивает растения но и собирает большой пул информации, которые впоследствии можно
использовать для каких-либо статей

**HydroLab Alpha отвечает на вопросы:**

* Сколько воды растение действительно потребляет?
* Как изменяется потребление со временем?
* Что происходит с питательным раствором?

### Сбор данных

* Измерение уровня воды (гидростатические датчики)
* Учёт расхода (включая двунаправленный поток)
* Оценка потребления воды и питательных веществ
* Высокоточное измерение PH и PPM
* Оценка загрязнения раствора органическими примесями

###  Аналитика и визуализация

* Логирование всей доступной телеметрии
* Графики через веб-интерфейс
* Широкие возможности анализа - отношение потребления воды к потреблению удобрений, остатка удобрений к PH
* Событийные логи

### Калибровка

* Произвольный размер питательного бака через калибровку
* Калибровка PPM датчиков буферными растворами
* Настройка через веб-интерфейс

### Управление

* Автоматизация насосов
* Сценарии полива
* Защита (перелив, сухой ход)
* Модульная архитектура управления

### Веб-интерфейс

* Мониторинг в реальном времени
* Ручное управление системой
* Инструменты калибровки
* Просмотр и анализ данных

---

## Архитектура

Система разделена на два уровня:

### Блок управления

* Raspberry Pi
* Сервер + веб-интерфейс
* Обработка данных и хранение логов

### Блок взаимодействия

* Микроноды на базе ESP32 с возможностью OTA обновления через блок управления
* Работа с датчиками и исполнительными устройствами
* Низкоуровневый ввод/вывод
